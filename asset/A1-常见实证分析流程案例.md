Y：代表被解释变量
X：代表核心解释变量
$controls：代表定义全局变量的全部控制变量
IV: 代表工具变量
M: 代表机制变量


* 定义全局数据和回归结果存放路径
```stata
global dataPath "C:\Users\86139\.claude\skills\STATA数据回归分析和结果撰写\data"
global resultPath "C:\Users\86139\.claude\skills\STATA数据回归分析和结果撰写\results"
```

* 切换路径并导入数据
```stata
cd $dataPath
use data.dta,clear
cd $resultPath
```

* 定义控制变量的全局变量
```stata
global controls age size Lev
```

* 设置面板数据
```stata
xtset id year
```

* 描述性统计
```stata
logout,save(描述性统计.rtf) word replace:tabstat Y X $controls,stat(N mean sd p50 min max) c(stat)
```

* 基准回归分析
```stata
reg Y X ,vce(cluster)
est store m1
reg Y X $controls,vce(cluster)
est store m2
reghdfe Y X $controls,absorb(id) vce(cluster)
est store m3
reghdfe Y X $controls,absorb(id year) vce(cluster)
est store m4
esttab m1 m2 m3 using 基准回归.rtf, r2 ar2  scalars(F)  b(%12.3f) se(%12.3f) star(* 0.1 ** 0.05 *** 0.01) replace nogap 
```

* 稳健性分析

```stata
** 替换被解释变量
reghdfe Y1 X $controls,absorb(id year) vce(cluster)
est store m1
reghdfe Y2 X $controls,absorb(id year) vce(cluster)
est store m2

** 替换解释变量
reghdfe Y X1 $controls,absorb(id year) vce(cluster)
est store m3

** 考虑滞后效应
reghdfe Y L.X $controls,absorb(id year) vce(cluster)
est store m4
esttab m1 m2 m3 m4 using 稳健性分析1.rtf, r2 ar2 scalars(F) b(%12.3f) se(%12.3f) star(* 0.1 ** 0.05 *** 0.01) replace nogap order(X X1 L.X $controls)

** 考虑极端值
preserve
    winsor2 Y X $controls, cuts(1 99) replace 
    reghdfe Y X $controls,absorb(id year) vce(cluster)
    est store m1
    esttab m1 using 考虑极端值.rtf, r2 ar2 scalars(F) b(%12.3f) se(%12.3f) star(* 0.1 ** 0.05 *** 0.01) replace nogap
restore 

```

* 内生性分析

** 工具变量法
```stata
reghdfe X IV $controls,absorb(id year) vce(cluster)
est store m1

ivreghdfe Y (X=IV) $controls,absorb(id year) r
est store m2 
esttab m1 m2 using 内生性分析.rtf, r2 ar2 scalars(F idstat rkf) b(%12.3f) se(%12.3f) star(* 0.1 ** 0.05 *** 0.01) replace nogap order(IV X $controls)
```

* 异质性分析

** 企业所有制异质性分析
```stata
reghdfe Y X $controls if soe == 1,absorb(id year) vce(cluster)
est store m1
reghdfe Y X $controls if soe == 0,absorb(id year) vce(cluster)
est store m2
esttab m1 m2 using 企业所有制异质性分析.rtf, r2 ar2 scalars(F idstat rkf) b(%12.3f) se(%12.3f) star(* 0.1 ** 0.05 *** 0.01) replace nogap
```

* 机制分析

** 
```stata
reghdfe Y X $controls ,absorb(id year) vce(cluster)

reghdfe M X $controls ,absorb(id year) vce(cluster)

```
